Michele Tobias

2024-09-30

Overview

0.1 Description

In this workshop, participants will learn about commonly used spatial data models and file formats - their structures and when to use any given option.

0.2 Learning Goals

By the end of this workshop, participants will understand

  • The difference between vector and raster data, as well as some niche data models
  • The reasons to use different data models
  • The underlying structure of each data model, conceptually

0.3 Prerequisites

This workshop is intended for beginners and is presented as a lecture with demonstrations of data. There are no prerequisites for this workshop and no software to install.
